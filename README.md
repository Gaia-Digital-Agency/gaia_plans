# Gaia Plans Presentation

This repository contains the Gaia Digital Agency operating model presentation and its supporting reference documents.

## Contents

- `index.html`: Main presentation page with embedded document popup viewer.
- `docs/ai_business_plan.html`: Rendered business plan document.
- `docs/ai_business_plan_appendix.html`: Appendix reference document.
- `docs/qa_automation.html`: Rendered QA automation reference.
- `docs/figma_to_site_automation.html`: Rendered Figma-to-site automation reference.
- `docs/gaia_plans.md`: Source planning document excluded from HTML conversion.

## Usage

Open `index.html` in a browser to view the presentation and launch the reference documents from the popup viewer.
